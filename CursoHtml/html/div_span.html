Divs y Spans en HTML

Sección 1

Este es un ejemplo de cómo utilizar spans dentro de divs para aplicar estilos específicos a partes del texto.

Sección 2

Los divs se utilizan para agrupar y estructurar contenido, mientras que los spans son elementos en línea que se utilizan para aplicar estilos a partes de un texto sin alterar la estructura del documento.
